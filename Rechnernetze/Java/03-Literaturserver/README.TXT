Literaturserver, der eine Gedichte, die in FOrm von Textdateien im Order
Literatur abgelegt sind, auslesen und verschicken kann. Das Protokoll erlaubt
die Nutzerauthentifizierung, die Navigation durch den Dateibaum und das
zeilenweise verschicken der Texte.

Verwendet die Abiturklasse Server.

Die Hilfsklasse FileSystem erlaubt die einfache Interaktion mit dem Dateisystem
unter Verwendung der Abiturklasse List.
