Einfache Version des Spiels "Stein-Schere-Papier" (oder
"Schnick-Schnack-Schnuck"), das über die Kommandozeile gespielt wird.

In diesem Projekt werden die Verwendung der Abiturklassen Server und Client
geübt, mit denen bidirektionale TCP-Verbindungen implementiert werden können.
